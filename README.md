# BTA-508
Instalador para adaptador Bluetooth Orico BTA-508 no Linux


## Instalação

Em seu terminal, rode o comando abaixo:

```
cd /tmp && git clone https://github.com/thespation/BTA-508 && chmod 755 BTA-508/* -R && cd BTA-508/ && ./instalar.sh
```
